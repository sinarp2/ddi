## 3. 난이도 기준

| 난이도 | 비중 | 설계 원칙 |
|--------|------|-----------|
| **L1** | 20% | 단일 규칙·명시적 정보 파악. 해당 도메인에만 관한 문항으로 범위를 명백히 한정. |
| **L2** | 50% | 복합 규칙·문맥적 상호작용. 2개 이상 도메인 융합 가능. 1단계 인과 추론 요구. |
| **L3** | 30% | 고차원 추론·외부 지식 통합. 3단계 이상 연쇄추론(CoT) 또는 지문 내 모순 탐지. |

## 4. 문항 유형별 출제 규칙

### 4-1. 객관식(MCQ)

- **5지 선다형** 원칙, 정답 수는 5개 중 **2개 이하**
- 지문 길이: L1 → 1~3문장, L2~L3 → 3~5문장 이상
- 발문 종결: `~고르시오.` / `~찾으시오.`
  - 최선답형: "가장 적절한 것을 **고르시오**."
  - 합답형(선택지 수 동일): "~을 〈보기〉에서 **고르시오**."
  - 합답형(선택지 수 다름): "~을 〈보기〉에서 **있는 대로 고르시오**."
  - 완성형: "(가), (나)에 들어갈 말로 옳은(적절한/가장 적절한) 것을 **고르시오**."
- 선택지는 길이·논리 순서 배열, **단서 노출 금지**
- 오답지: 정답과 유사하나 미세한 뉘앙스 차이·잘못된 높임·부적절한 접속 포함

#### 난이도별 오답 설계 전략 (예시: 경어법)

| 난이도 | 오답 전략 |
|--------|-----------|
| L1 | 규범에서 명백히 벗어난 형태 (주체 높임 상황에 낮춤 표현) |
| L2 | 문법은 맞으나 문맥상 어색 (격식체·비격식체 혼용, 부적절한 접속사) |
| L3 | 의미는 통하나 사회적 거리감·사용역에 미세하게 어긋남 (간접높임 남용, 맥락적 오용) |

---

### 4-2. 단답형

- 정답이 단어 또는 구(phrase) 단위로 고정되는 경우 적용
- **허용 변이 최소 3개** 이상 지정 (띄어쓰기·문장부호 차이 포함)
- 빈칸 완성형: 앞뒤 문장 성분(부사어·서술어 호응)으로 강하게 제약
- 교정형: "다음 문장에서 어색한 부분을 한 단어로 고쳐 쓰시오." — 수정 전후 언어적 일관성 유지

---

### 4-3. 서술형(개방형)

- **시나리오 기반 설계**: 화자·청자·장소·목적 명시
- **복합 제약 조건** 필수 부여:
  - 형식 제어: 문체(하십시오체·평어체 등), 글자 수 제한
  - 내용 제어: 필수 키워드 3개 내외 포함
  - 금지 제어: 특정 단어 사용 금지, 번역투 사용 금지
- **할루시네이션 방지 지침 명시**: 지문에 없는 사실 임의 서술 금지

## 5. 문항 개발 양식 (메타데이터 작성 필수)

각 문항은 아래 항목을 **반드시** 작성합니다.

| 항목 | 작성 내용 |
|------|-----------|
| **문항 ID** | 도메인-난이도-일련번호 (예: `REG-L3-0001`) |
| **개발자명** | 출제자 성명 |
| **평가 도메인** | 해당 도메인 선택 |
| **난이도** | L1 / L2 / L3 |
| **문항 유형** | 선택형(MCQ) / 단답형 / 서술형(개방형) |
| **상황(scenario)** | (서술형·대화문 필수) 맥락, 화자·청자 관계, 장소, 목적 |
| **입력 텍스트** | AI에게 제시할 원문 텍스트 |
| **발문** | 예: "가장 적절한 것을 고르시오." |
| **수행 조건** | (서술형 필수) 문체, 키워드, 금지어, 글자 수 |
| **선택지** | (선택형 필수) ①②③④⑤ |
| **정답 및 해설** | 정답 번호 또는 모범 답안 + 언어학적 해설 |
| **허용 변이 형태** | (단답형·서술형 필수) 허용 답안 목록 및 가중치(0.15~1.00) |
| **정답 해설** | 어문 규정·담화 이론 근거 상세 설명 |
| **채점 루브릭** | (서술형 필수) 항목별 배점 및 1~5점 척도 수행 기준 |
| **채점 항목 가중치** | 언어적 정확성 [ ] / 분석 및 논리성 [ ] / 전략 및 적절성 [ ] / 조건 및 제약 준수 [ ] |
| **금지어 목록** | 답안 작성 시 사용 금지 단어 |
| **출처** | 원천 데이터 출처 (뉴스, 국립국어원 말뭉치 등) |

## 6. 서술형 채점 루브릭 기준

### 6-1. 범용 루브릭 (5점 척도)

| 평가 항목 | 5점 | 3점 | 1점 |
|-----------|-----|-----|-----|
| **언어적 정확성** | 맞춤법·띄어쓰기·문장성분 호응 완벽 준수 | 부차적 오탈자·띄어쓰기 오류 1~2개 | 비문 다수, 규범 위반 심각 |
| **분석 및 논리성** | [메타 문항] 오류 원인을 어학적·담화적 근거로 명확히 분석 | 오류 지점은 정확하나 논거 부족·전문 용어 부적절 | 오류 판단 불가 또는 틀린 근거 |
| **전략 및 적절성** | [생성 문항] 화행·레지스터·경어법 완벽 통제 | 상황에 부합하나 어휘·문체에 미세한 부조화 | 문체 전혀 안 맞거나 화행 목적 달성 실패 |
| **조건 및 제약 준수** | 필수 키워드·금지어·분량·문장구조 모든 제약 완벽 이행 | 제약 하나 누락 또는 금지어 유사 형태 노출 | 제약 다수 무시·금지어 그대로 사용 |

### 6-2. 도메인별 가중치

| 문항 목적/도메인군 | 언어적 정확성 | 분석·논리성 | 전략·적절성 | 조건·제약 준수 |
|-------------------|-------------|------------|------------|---------------|
| 범용 서술형 | 0.25 | 0.25 | 0.25 | 0.25 |
| 문법 규범 판단 중심 | 0.35 | 0.35 | 0.10 | 0.20 |
| 담화·표현 적절성 중심 | 0.25 | 0.15 | 0.35 | 0.25 |
| 추론·무형대용어 복원 | 0.20 | 0.40 | 0.15 | 0.25 |
| 문맥 이해도·화시 | 0.20 | 0.30 | 0.25 | 0.25 |

## 8. 품질 기준

### 반드시 준수

- [ ] 정답의 근거가 지문 내에 **명시적으로 포함**
- [ ] 전문가 집단에서 **이견 없는 유일한 정답**
- [ ] 선택지 간 **간섭 없음**
- [ ] 단서가 선택지 형태·조건에서 **읽히지 않음**
- [ ] 인터넷 공개 자료 단순 인용 **금지** (100% 신규 창작 원칙)
- [ ] 외부 자료 인용 시 **70% 이상 실질적 개작**
- [ ] 혐오·차별 표현 **철저 배제**
- [ ] AI 생성 문항 **절대 사용 금지**

### Google-Proof 설계

단순 지식 검색·암기 데이터로 해결 불가능한 복합 추론·비전형적 맥락 이해 문항 개발

### 저작권 안전

공공누리(KOGL), 정부 보도자료, 관보, 법령, 저작권 만료 고전문학·사료 적극 활용  
출처는 **메타데이터에 내부 기록** 필수

## 9. 교차 검수 절차

1. 출제자가 문항 완성 후 **자기 검수**
2. 도메인 내 **전문가 2인 교차 검토** (별도 검수팀)
3. 검수 항목:

| 구분 | 항목 |
|------|------|
| **형식 검수** | 문항 규격, 선지 배열, 단서 차단 여부, 난이도별 오답 설계 규격, 단답·서술형 작성 조건, 채점 루브릭 규격 준수 |
| **내용 검수** | 지문 근거 완결성, 정답 유일성, 단답형 허용 답안·서술형 예시 답안 타당성, 선지 간 간섭, 오답지 매력도 |

## 10. 제출 형식

- **파일 형식**: Excel (.xlsx) 또는 JSON (별도 양식 참조)
- **문항 ID 규칙**: `[도메인코드]-L[1/2/3]-[4자리 일련번호]`
  - 예: `REG-L2-0001` (경어법 L2 1번)
- **제출 단위**: 도메인별 일괄 제출
- **모범 답안**: 서술형은 반드시 포함 (채점 유연성 확보 목적)
